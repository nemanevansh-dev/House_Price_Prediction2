# House Price Prediction 

##  Problem
Predict house prices based on features like location, area, etc.

##  Solution
Built a Machine Learning model using Linear Regression with data preprocessing and feature scaling.

##  Tech Stack
- Python
- Pandas
- Scikit-learn
- Flask

##  Features
- Real-time house price prediction
- User input via web interface
- Deployed ML model
